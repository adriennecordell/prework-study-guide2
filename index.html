HTML

  • HTML is made up of elements which work together to provide content and basic framework for all pages on the web.
  • The head element contains information or metadata that helps the browser render the page correctly.
  • The body element contains content that is shown to the user.
  • All elements we want the user to see and interact with must be placed inside the body element.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • Lang="en" tells the browser the language for the webpage.

CSS

  • There are three ways to style a webpage using CSS; Inline CSS, Internal CSS, and External CSS.
  • An example of inline CSS is using the 'style' attribute to change the color of a single h1 heading.
  • An example of Internal CSS is using the style attribute to change the color of all the h1 headings.
  • External CSS, also the most common way to style a webpage, is used by linking a file in the 'link' element placed in the HTML's 'head'
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want sound the content inside an element.

Git

  • Git command must precede every Git statement.
  • The Git status command tells us which branch we are currently working in.
  • The Git checkout command switches branches.
  • Git add -A command adds modifications in current working branch to the staging area and the -A flag indicates that we want to add all changes.
  • Git commit -m "add starter code" is a commit. The words in quotations describe what the commit contains.
  • Commits must be in quotations!

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Each line of JavaScript code must end in a semicolon.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a single varible that is used to hold a group of data. Usually this data is related in some way.
  • A for loop contains three inportant statements. The first statement determines the starting point for our loop. The second statement determines the condition of the loop. The third statement is what allows the array to iterate over each time.
  • To create a function we use the 'function' keyword. A functions name is followed by parenthesis.
